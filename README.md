## 📄 프로젝트 소개
- Findex는 외부 Open API와 연동하여 금융 지수 데이터를 제공하는 대시보드 서비스입니다.
- 프로젝트 기간: 2025.04.21 ~ 2025.04.29

## 🍃 기술 스택
- Backend: Spring Boot, Spring Data JPA
- Database: PostgreSQL
- 공통 Tool: Git & Github, Discord, Notion


## 💻 팀원별 구현 기능 상세
🟢 **황지환**
~~~text
1. 연동 작업 관리
    - 연동 정보 분석
    - 지수 정보 연동
    - 지수 데이터 연동
    - 연동 작업 목록 조회
2. 자동 연동 설정 관리
    - 배치에 의한 지수 데이터 연동 자동화
~~~
